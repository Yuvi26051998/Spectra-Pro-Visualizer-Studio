# Spectra-Pro-Visualizer-Studio
Spectra is a professional-grade, real-time audio visualizer built with the Web Audio API and HTML5 Canvas. It transforms music into stunning, customizable visual animations with studio-quality controls. Perfect for content creators, musicians, video producers, and visual artists who want to create engaging audio-reactive visuals for social media, music videos, live performances, and streaming overlays.

**_**Key Features**

Audio Processing & Playback

Supports MP3, WAV, OGG, and other standard audio formats

Built-in demo tone for instant testing

Drag-and-drop support for seamless file loading

Full playback controls with pause/resume functionality

Audio trimming with start/end position sliders

**7 Visualizer Shapes**

Bars — Classic frequency bars that pulse to the beat

Wave — Smooth oscilloscope-style waveform with layered strokes

Circle — Radial frequency display with expanding rings

Particles — Dynamic particle system that explodes to the rhythm

Kaleido — Kaleidoscopic geometric patterns that shift with the music

Spectrogram — Frequency heatmap visualization

Radial — Circular particle burst effect

Complete Customization

Logo Upload — Add your own image with zoom control (30%–200%)

Artist & Track Names — Editable text with independent color pickers

Position Controls — X/Y sliders to place text anywhere on canvas

7-Color Palette — Choose custom colors that react to audio intensity

Reactivity Slider — Control sensitivity (0.2x – 2.0x)

Particle Density — Adjust particle count for performance tuning

Glow Intensity — Add professional glow effects to visualizer elements

**Background Media**

Upload static images (JPG, PNG, GIF)

Upload MP4 videos as animated backgrounds

Drag-and-drop support for quick swapping

Preset System

Five built-in presets: Default, Neon, Sunset, Ocean, Midnight

Export/import custom presets as JSON files

**Export Capabilities**

Export PNG images of your visualizer

Record MP4 videos (WebM format) with audio synchronization

Save and load complete preset configurations

**Interactive Controls**

Keyboard shortcuts: 1-7 for shape switching, Space for play/pause

Live audio meter with visual feedback

Real-time updates on all settings

**Technical Specifications**

FFT size 1024 for detailed frequency analysis

60fps smooth animation with high DPI support

Responsive design adapts to any screen size

Optimized for real-time rendering with GPU acceleration

Spectra turns your music into visual art. Whether you're creating content for YouTube, Instagram, TikTok, or live performances, this tool gives you professional-grade controls in an intuitive interface. With presets, exports, and real-time customization, you can create stunning audio-reactive visuals in minutes — no coding required.
